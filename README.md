# Persona AI

> Chat with AI-powered versions of Hitesh Choudhary and Piyush Garg.

🔗 **Live Demo**: https://persona-ai-[your-hash].vercel.app

## Tech Stack
Next.js 15 · TypeScript · Groq · Vercel AI SDK · Tailwind CSS · shadcn/ui

## Features
- Believable AI personas with deep behavioral prompting
- Streaming responses with Markdown + syntax highlighting
- Text-to-speech with Hinglish transliteration
- Persona-specific accent theming
- Token-aware conversation memory
- Rate limiting per persona
